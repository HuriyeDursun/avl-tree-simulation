# AVL Tree Simulation (Python & Tkinter)

This project is an **interactive AVL Tree simulation** developed using **Python** and **Tkinter**.  
It visually demonstrates how an AVL Tree maintains balance through rotations during **insertion** and **deletion** operations.

The application is designed for **educational purposes**, helping students understand:
- Self-balancing binary search trees
- AVL rotations (LL, LR, RR, RL)
- Tree traversals
- Height and balance factor concepts

## ğŸ® How to Use

- Enter an integer value in the input box.

- Use the buttons to:
  - **Insert**: Add a value to the AVL Tree  
  - **Delete**: Remove a value from the tree  
  - **Search**: Find a value and see its height and balance factor  

- Use traversal buttons to display:
  - Inorder  
  - Preorder  
  - Postorder  

- Click **Sample Tree** to automatically generate a predefined AVL Tree.

- Click **Clear** to reset the tree.

Rotation messages will be displayed automatically when rebalancing occurs.
